# Simple CRUD with CI4 & Mysql

## Installation

Clone Project

```bash
  git clone https://github.com/attmhd/publikasi_ci4.git
```

Move to Project

```bash
  cd publikasi_ci4
```

Rename env copy to .env

Import publikasi.sql in phpMyadmin

Install all libraries needed

```bash
  composer install
```

Run Project

```bash
  php spark serve
```
